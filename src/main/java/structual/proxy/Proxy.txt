Шаблон: Прокси
==============================================

Цель:

Для замещения другого обьекта и контроля доступа к нему.
================================================

Для чего используется:

Для обеспечения контроля доступа к определенному обьекту.

=================================================
Пример использования:

-когда есть необходимость ссылаться на обьект отличным от обычного указателя

    * удаленное замещение - локальный представитель вместо обьекта в другом
                            адресном пространстве.

    * вирутальное замещение - создание "тяжелых обьектов" при необходимости

    * защищенное замещение - контроль доступа к указанному обьекту

    * "умная" ссылка -   замена обычного указателя
                        (подсчет числа ссылок, установка блокировки т.д)
